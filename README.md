# Robotica-VizDoom
## Contribución
Replicamos el experimento de VizDoom: Una plataforma de investigación de IA basada en Doom para el Aprendizaje por Refuerzo Visual.

## Tecnologías Usados
1. Package vizdoom 1.1.13
2. PyCharm 2021.3.1
